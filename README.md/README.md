# chatbot-backend

## Project setup

mkdir chatbot-backend
cd chatbot-backend
npm init -y

## dependencies
npm install express axios dotenv cors

# .env
.env: Environment variables configuration file.

REPLICATE_API_TOKEN=your_replicate_api_token

# compile the server side
node index.mjs

index.mjs : Main server file.
